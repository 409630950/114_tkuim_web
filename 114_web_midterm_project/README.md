# MyPet 喵喵生活助手

## 組員

## 專案簡介
MyPet 是一個以貓咪為主題的互動生活小工具，整合了寵物健康、飼料建議與疫苗提醒等功能。  
使用者可註冊登入並記錄愛貓資訊，系統會根據品種、體重與活動量自動提供餵食克數與疫苗時程建議。  
同時具備歷史紀錄、夜間模式與表單驗證，打造溫柔可愛又實用的寵物管理網頁。

## 使用技術
- **HTML5**：語意化結構與內容分區  
- **CSS3 / Bootstrap 5**：RWD 響應式設計與統一排版  
- **JavaScript (ES6)**：互動行為、即時計算與 LocalStorage 儲存  
- **Constraint Validation API**：表單驗證與即時提示  
- **DOM 操作**：`querySelector`、`addEventListener`、`createElement`等動態更新

## 功能特色
1. 會員系統：支援註冊 / 登入 / 登出，含密碼強度檢測  
2. 寵物健康分析：根據品種與體重顯示健康狀態（過輕 / 正常 / 過重）  
3. 飼料克數建議：依據活動量與熱量計算每日建議克數  
4. 疫苗提醒功能：自動計算下次 FVRCP、狂犬病施打日期  
5. 歷史紀錄系統：可手動儲存飼料與疫苗建議，方便回顧  
6. 夜間模式：主題切換、顏色自動適配，減少夜間刺眼度  
7. RWD 介面：各裝置皆能自適應顯示，介面柔和可愛
